скачать ollama из https://ollama.com/download

открыть консоль и ввести следующие команды:

# для проверки установленно или нет
ollama --version

# установка модели
ollama pull phi3

# запуск сервера
ollama serve


ТАКЖЕ поменять конфигурацию OLLAMA_HOST = "0.0.0.0"
можно выбирать место установки приложения (конец статьи) и модели (примерно середина)
https://www.glukhov.org/post/2024/06/move-ollama-models/

ВАЖНОЕ ПРИМЕЧАНИЕ:
нужно в консоли сначала выполнить ollama serve, а уже потом запускать само приложение ollama, иначе будет ругаться,
что порт уже занят

ПО ПОВОДУ ПОДДЕРЖКИ НА НАШЕМ САЙТЕ С OLLAMA
сообщение отправляется, надо просто подождать и прийдёт alert на сайте
